# 🚀 My First PR

## 🎯 Goal
Make your first open-source contribution in less than 10 minutes.

## 📌 What You Will Do
Add your name to the CONTRIBUTORS.md file.

## ⚙️ Prerequisites
- Git installed
- GitHub account

## 🪜 Steps

1. Fork this repository
2. Clone your fork
3. Create a new branch
4. Add your name
5. Commit your changes
6. Push to GitHub
7. Create a Pull Request

## 🎉 Outcome
Congratulations! You just made your first PR 🚀

## ❗ Common Errors

| Error | Fix |
|------|-----|
| authentication failed | Use Personal Access Token |
| not a git repository | Run `cd my-first-pr` |
| updates rejected | Run `git pull origin main` |
